# A 股股票代码规则说明

## 1. 代码格式总览

A 股股票代码为 6 位数字，根据上市交易所和板块有以下规律：

| 代码段 | 交易所 | 板块 |
|-------|-------|-----|
| 60xxxx | 上交所（SH） | 沪市主板 |
| 688xxx | 上交所（SH） | 科创板 |
| 000xxx | 深交所（SZ） | 深市主板 |
| 001xxx | 深交所（SZ） | 深市主板（新） |
| 002xxx | 深交所（SZ） | 中小板（已合并入主板） |
| 003xxx | 深交所（SZ） | 深市主板（新） |
| 300xxx | 深交所（SZ） | 创业板 |
| 301xxx | 深交所（SZ） | 创业板（新） |
| 4xxxxx / 8xxxxx | 北交所（BJ） | 北交所 |

## 5. 判断市场归属（代码 → market 参数）

```python
def get_market(code: str) -> str:
    """根据代码判断市场（用于 akshare 资金流向接口）"""
    code = code.strip()
    if code.startswith("6") or code.startswith("9"):
        return "sh"
    elif code.startswith("0") or code.startswith("3"):
        return "sz"
    elif code.startswith("4") or code.startswith("8"):
        return "bj"
    return "sh"
```

## 6. 注意事项

- **退市股票**：历史代码可能被新股复用，获取历史数据时注意时间范围
- **ST 股票**：代码不变，名称前缀变为 `*ST` 或 `ST`，可通过名称字段识别
- **停牌股票**：AKShare 可能返回空数据或缺失部分交易日，脚本需做判断
- **北交所股票**：AKShare 对北交所的支持相对有限，数据可能不完整
- **指数 vs 个股**：两者使用不同的 AKShare 接口，不可混用
