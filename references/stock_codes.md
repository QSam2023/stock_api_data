# A 股股票代码规则

## 代码格式

A 股股票代码为 **6 位数字**，根据首位数字区分交易所和板块：

| 首位 | 交易所 | 板块 | 示例 |
|------|--------|------|------|
| 6 | 上交所（SH） | 主板 | 600519（贵州茅台） |
| 9 | 上交所（SH） | B 股 | 900901 |
| 0 | 深交所（SZ） | 主板 | 000001（平安银行） |
| 2 | 深交所（SZ） | B 股 | 200002 |
| 3 | 深交所（SZ） | 创业板 | 300750（宁德时代） |

## 带前缀格式

部分场景使用带市场前缀的格式：

- 上交所：`sh600519`
- 深交所：`sz000001`

前缀不区分大小写（`SH600519` = `sh600519`）。

## 指数代码

| 代码 | 名称 |
|------|------|
| 000001 | 上证指数（注意：与平安银行代码相同，需通过上下文区分） |
| 399001 | 深证成指 |
| 399006 | 创业板指 |
| 000300 | 沪深 300 |
| 000016 | 上证 50 |
| 000905 | 中证 500 |

**注意**：本项目当前仅支持个股分析，暂不支持指数。指数数据获取需使用不同的 AKShare 接口（`stock_zh_index_daily`）。
