将经典投机大师的方法论融入到程序化分析中，是开发金融数据 Skill 最核心的价值所在！普通的指标（如单纯的 MACD、KDJ）容易出现滞后或假信号，而大师们的方法论往往更关注**“量价关系（Volume & Price）”、“趋势突破（Breakout）”和“相对强弱（Relative Strength）”**。

为了方便你后续将这些理论转化为 Python 代码（pandas / pandas-ta），我为你提炼了 5 位最具代表性的投机大师的核心方法论，以及如何将它们转化为可计算的技术指标：

1. 尼古拉斯·达瓦斯 (Nicolas Darvas) —— 【箱体理论 Box Theory】

达瓦斯是一名舞蹈家，靠这套理论在股市赚了 200 万美元。他的核心逻辑是“动能与趋势的突破”。

• 核心理念：股票在上涨过程中会形成一个个“箱体”（震荡区间）。当股价放量突破当前箱体顶部时买入，跌破箱体底部时止损。
• 可编码的指标设计：
  • 计算上轨（Box Top）：过去 N 天（如 20 天）的最高价，且最近 3 天没有创出新高（确认箱顶）。
  • 计算下轨（Box Bottom）：确认箱顶后的最低价，且最近 3 天没有创出新低。
  • 交易信号：Close > Box Top 且当日成交量（Volume）> 过去 50 日平均成交量的 1.5 倍（放量突破）。

2. 威廉·欧奈尔 (William O'Neil) —— 【CAN SLIM 与相对强弱】

欧奈尔是成长股投资大师，他不仅看基本面（CAN），极度看重技术面（SLIM）中的趋势和市场地位。

• 核心理念：永远不要在下跌趋势中买股票，只买市场中最强势的股票（强者恒强）。
• 可编码的指标设计：
  • RPS（相对强弱指标 Relative Price Strength）：这是欧奈尔的灵魂指标。计算某只股票过去 120 个交易日（半年）或 250 个交易日（一年）的涨幅，与整个 A 股市场所有股票进行排名。只选取排名前 80% 甚至 90% 的股票。
  • 均线多头排列：收盘价 > 50日均线 > 200日均线，且 200 日均线必须是向上的（过滤掉长期下跌的股票）。
  • 杯柄形态（Cup and Handle）的量价特征：突破前期高点时，成交量必须比平时大 50% 以上；而在前期回调（洗盘）时，成交量必须显著萎缩。

3. 理查德·威科夫 (Richard Wyckoff) —— 【VSA 量价分析】

威科夫与道氏、江恩齐名，但他更贴近交易本质，他的理论是华尔街机构交易员的必修课。

• 核心理念：价格是表象，成交量是内因。主力资金的建仓（吸筹）和出货（派发）都会在量价的异常中留下痕迹。
• 可编码的指标设计：
  • 努力没有结果 (Effort vs Result)：例如，当日成交量极大（Effort很大），但 K 线的实体非常小（或者是长上影线）。用代码写就是：Volume > 2 * 均量 且 (High - Low) / Close < 2%。在顶部出现意味着主力在派发（出货），在底部出现意味着主力在吸筹。
  • 弹簧效应 (Spring/False Breakout)：股价跌破前期重要支撑位（如 60 日低点），但在 1-2 天内迅速收回支撑位之上，且伴随放量。这是典型的“主力挖坑洗盘”信号。

4. 斯坦·温斯顿 (Stan Weinstein) —— 【四阶段分析法】

温斯顿的《笑傲牛熊》是趋势交易者的圣经，尤其适合 A 股这种牛短熊长、板块轮动的市场。

• 核心理念：万物皆周期，股票分为四个阶段：筑底期、上升期、顶部震荡期、下降期。只在“第一阶段向上突破进入第二阶段”时买入。
• 可编码的指标设计：
  • 生命线（30周均线 / 约等于 150日均线）：
    • 拒绝买入信号：如果股价在 150 日均线下方，或者 150 日均线正在向下倾斜，绝对不买。
    • 启动信号：股价横盘震荡超过 3 个月（最高价和最低价的极差缩小），随后某天 Close 强势突破 150 日均线，且 150 日均线斜率由平转正。
  • 曼斯菲尔德相对强弱（Mansfield RS）：股票走势除以沪深300指数的走势，并计算其 52 周移动平均线。如果该数值 < 0，说明跑输大盘，直接过滤。

5. 杰西·利弗莫尔 (Jesse Livermore) —— 【关键点突破法】

华尔街的传奇大空头和多头，他的核心思想非常朴素但致命。

• 核心理念：不要抄底，不要猜顶。等待市场自己走出明确的“关键点”（Pivot Points），在市场确认趋势的那一刻上车。最小阻力位方向就是趋势方向。
• 可编码的指标设计：
  • 阻力最小路线法则：创历史新高（或阶段性比如 250 日新高）的股票，上方没有套牢盘，阻力最小。代码可以写：Close == max(High, 250天)。
  • 自然回撤法则：突破后允许正常回撤，但回撤幅度不能超过突破涨幅的 50%，且回撤时的平均成交量必须小于突破时的平均成交量（缩量回踩）。

💡 建议在 OpenClaw Skill 中的实战组合落地

既然我们要写 Python 脚本让 LLM 去调用，我建议为你的 A 股分析 Skill 设计这样一个 “三步走大师过滤模型”：

1. 第一步：温斯顿 & 欧奈尔的“大势与均线过滤”（防守）
  • 代码逻辑：只保留 Close > 150日均线 且 150日均线斜率 > 0 的票。这一步能帮你直接避开 A 股 80% 的深坑和阴跌股。
2. 第二步：威科夫的“量价健康体检”（透视）
  • 代码逻辑：检查最近 10 个交易日，上涨日的平均成交量是否大于下跌日的平均成交量。排除那些“放量下跌、缩量上涨”的主力出货图。
3. 第三步：达瓦斯 & 利弗莫尔的“关键点突破”（进攻信号）
  • 代码逻辑：识别过去的震荡箱体（比如最近一个月最高低点振幅 < 15%），当某天收盘价突破箱顶，且成交量翻倍时，LLM 输出强烈的异动提醒。

在 OpenClaw 中，我们可以写一个 scripts/masters_indicators.py，让大模型只需要传入股票代码（如 sz000001），脚本就自动计算上述的大师指标并返回 JSON 或文字报告，大模型再将其翻译成人话给用户。
